# 初始化 SiYuan CLI

## 背景

为思源笔记 (SiYuan Note) 开发一个命令行工具，参考 Obsidian CLI 的能力。思源笔记是一个隐私优先的个人知识管理系统，支持细粒度块级引用和 Markdown 所见即所得。

## 目标

- 提供完整的 CLI 界面操作思源笔记
- 支持笔记本、文档、块的核心操作
- 支持多种输出格式 (Table, JSON, Markdown)
- 遵循 SDD 开发规范

## 范围

### 功能范围

1. **笔记本管理**
   - 列出所有笔记本
   - 创建新笔记本
   - 重命名笔记本
   - 删除笔记本
   - 打开/关闭笔记本

2. **文档管理**
   - 创建文档
   - 删除文档
   - 重命名文档
   - 移动文档
   - 导出文档

3. **块操作**
   - 获取块内容
   - 更新块
   - 删除块
   - 插入块
   - 移动块
   - 获取子块

4. **搜索**
   - 执行 SQL 查询

5. **配置管理**
   - 初始化配置文件
   - 查看/设置配置

### 技术范围

- **平台**: 支持 Windows/Linux/macOS
- **语言**: TypeScript
- **运行时**: Node.js 16+
- **发布**: npm 包

## 约束

- 所有 API 调用需要 Token 认证
- API 端点默认为 http://127.0.0.1:6806
- 遵循思源笔记 API 规范
